Para Iniciantes

Se você está começando agora, aqui está uma explicação simples sobre como usar a biblioteca.

O que é Impact JS?

A biblioteca Impact JS serve para detectar quando dois elementos colidem, ou seja, quando eles se encontram fisicamente na tela. Isso é útil para criar animações e interações no seu site.

Como Usar?

  1. Adicionar a Biblioteca: Para carregar a biblioteca em seu projeto, use o fetch() para buscar o código diretamente do link abaixo e executar: 
    
fetch('https://raw.githubusercontent.com/cmstudioapps/Bibliotecas/main/impact.js')
    .then(response => response.text())
    .then(script => {
        eval(script);  // Executa o código da biblioteca
    })
    .catch(error => console.error('Erro ao carregar a biblioteca:', error));
  2. Verificar Colisões: Use a função detectar para verificar se há colisões entre elementos. 
    let elementos = [document.getElementById("objeto1"), document.getElementById("objeto2")];
detectar(elementos, (elem1, elem2) => {
    console.log("Colisão detectada entre " + elem1.id + " e " + elem2.id);
});
  3. Parar o Movimento: Você pode parar o movimento de um elemento ao detectar uma colisão, definindo o parâmetro parar como true. 
    detectar(elementos, (elem1, elem2) => {
    console.log("Parando movimento de " + elem1.id + " e " + elem2.id);
}, true);
  4. Alterar Interrupção de Movimento: Use a função setPararMovimento para ativar ou desativar dinamicamente a interrupção de movimento. 
    setPararMovimento(true); // Ativa
setPararMovimento(false); // Desativa

Para Programadores Experientes

Se você já é familiarizado com JavaScript, aqui está uma explicação mais detalhada sobre o funcionamento da biblioteca.

Visão Geral

A biblioteca Impact JS permite detectar colisões entre múltiplos elementos na página e interromper o movimento deles utilizando CSS (propriedades como animation, transition ou transform).

Funções Principais

  • Função detectar: Verifica colisões entre um conjunto de elementos e executa um callback para cada colisão. 
    detectar(elementos, (elem1, elem2) => {
    console.log(`Colisão detectada entre ${elem1.id} e ${elem2.id}`);
}, true);
  • Função colisao: Compara os limites de dois elementos e retorna se houve interseção (colisão). 
    const colidiu = colisao(elem1, elem2);
if (colidiu) {
    console.log("Os elementos colidiram!");
}
  • Função pararMovimento: Remove qualquer animação ou transformação CSS do elemento. 
    pararMovimento(elem1); // Interrompe o movimento do elemento
  • Função setPararMovimento: Ativa ou desativa a interrupção do movimento de elementos colididos. 
    setPararMovimento(true); // Ativa
setPararMovimento(false); // Desativa
