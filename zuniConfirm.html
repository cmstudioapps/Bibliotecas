Documentação - zuniAlertConfirm

Bem-vindo à documentação da biblioteca zuniAlertConfirm!

Essa biblioteca permite criar alertas de confirmação com opções personalizadas, como "Sim" e "Não". O usuário pode escolher uma opção e a ação correspondente será executada.

Como Usar

Inclua o arquivo JavaScript da biblioteca no seu projeto e chame a função zuniAlertConfirm passando as opções desejadas.

<script src="caminho/para/zuniAlertConfirm.js"></script>

Exemplo de Código

Abaixo está um exemplo de como usar a função zuniAlertConfirm.


zuniAlertConfirm({
  title: 'Deseja continuar?',
  message: 'Essa ação não pode ser desfeita.',
  confirmText: 'Sim',
  cancelText: 'Não',
  backgroundColor: '#28a745',
  confirmCallback: function () {
    alert('Você confirmou!');
  },
  cancelCallback: function () {
    alert('Você cancelou!');
  },
});

Funções e Parâmetros

Esta biblioteca possui os seguintes parâmetros:

  • title: O título do alerta (opcional).
  • message: A mensagem a ser exibida (obrigatório).
  • confirmText: O texto do botão "Sim" (opcional).
  • cancelText: O texto do botão "Não" (opcional).
  • backgroundColor: A cor de fundo do botão "Sim" (opcional).
  • confirmCallback: Função chamada quando o usuário clica em "Sim" (opcional).
  • cancelCallback: Função chamada quando o usuário clica em "Não" (opcional).
